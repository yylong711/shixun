### api

- url格式api/jd 请求的是jd字段

通过传入参数进行解析
传入三个参数
 - id为分词器序号 
 - sentence=句子内容 
 - page为请求的页号，每页返回五条数据

url: http://127.0.0.1:5000/api/jd/?id=1&sentence=%E5%8F%91%E5%B8%83%E4%BC%9A&page=2
